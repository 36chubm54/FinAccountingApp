    """–°–µ—Ä–≤–∏—Å –∫–æ–Ω–≤–µ—Ä—Ç–∞—Ü–∏–∏ –≤–∞–ª—é—Ç –≤ –±–∞–∑–æ–≤—É—é (KZT)."""

        """
        Args:
            rates: –°–ª–æ–≤–∞—Ä—å –∫—É—Ä—Å–æ–≤ {–∫–æ–¥_–≤–∞–ª—é—Ç—ã: –∫—É—Ä—Å_–∫_–±–∞–∑–æ–≤–æ–π}

        self._base = base

        """–ö–æ–Ω–≤–µ—Ä—Ç–∏—Ä—É–µ—Ç —Å—É–º–º—É –≤ –±–∞–∑–æ–≤—É—é –≤–∞–ª—é—Ç—É."""
